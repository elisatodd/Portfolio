Flamingo Game Engine

C++ based video game engine developed in 2022, alongside five other students.
Flamingo is a basic game engine crafted to learn the bases of game programming, focusing on 3D rendering and data driven games.
It’s designed to support three dimensional RPG games, featuring a main player avatar controlled by user input.

Development Highlights


  • 🎮 Programmed in C++
  • 📋 Data driven engine
  • 🌉 LUA for scripting via LuaBridge
  • 🧌 Rendering with Ogre
  • 🔫 Bullet for physics
  • ✏️ SDL for input
