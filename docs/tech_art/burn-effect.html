Burn Expansion Shader

This project showcases a dynamic burning effect implemented in Unreal Engine 5 using Runtime Virtual Textures (RVT). When a projectile hits the terrain, a Blueprint actor is spawned that controls an expanding burn area. The expansion is fully parameterized, allowing precise control over the burn radius and growth speed.

The core shader acts as a mask that progressively reveals three distinct layers: burning, burnt, and dissolved, simulating the state of the terrain and the underlying foliage as the fire spreads. This shader is written into a RVT that the foliage and terrain materials read and interpret accordingly.

The effect combines shader programming with Blueprint logic to create a visually rich, performant, and customizable environment interaction.
